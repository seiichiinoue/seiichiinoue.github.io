Mathjax記法メモ

hugoで数式を書く際にはmathjaxを使用しているのですが，若干記法が違うらしく，読み込んでくれなかったためメモ．

経験上，.mdに直接数式を書こうとすると以下の対処が必要ぽい．

改行

改行は\\ではなく\\\\\\\で，バックスラッシュ6つ必要．

$$e^{i \theta} = cos \theta + i \; sin \theta \\\\\\
e^{i \pi} + 1 = 0 \quad if \; \theta = \pi$$

$$e^{i \theta} = cos \theta + i \; sin \theta \\
e^{i \pi} + 1 = 0 \quad if \; \theta = \pi$$

アンダーバー ( _ )

添字をつける際などアンダーバーを多用していると，たまにレンダリングがうまくいかない．

公式サイトによると，そういったバグはもう報告され済みだから以下の方法で対処してねーとのこと．

$${\frac{\partial v_v (t-q)}{\partial v_u (t)} = \sum_{l_1 = 1}^{n} \cdots　\sum_{l_{q-1} = 1}^{n} \prod_{m=1}^q f'_{l_m} (net_{l_m} (t - m)) w_{l_m l_{m-1}}}$$

これだとうまくレンダリングされないので，以下のように_を\_に変更する．

$${\frac{\partial v\_v (t-q)}{\partial v\_u (t)} = \sum\_{l\_1 = 1}^{n} \cdots　\sum\_{l\_{q-1} = 1}^{n} \prod\_{m=1}^q f'\_{l\_m} (net\_{l\_m} (t - m)) w\_{l\_m l\_{m-1}}}$$
$${\frac{\partial v_v (t-q)}{\partial v_u (t)} = \sum_{l_1 = 1}^{n} \cdots　\sum_{l_{q-1} = 1}^{n} \prod_{m=1}^q f'_{l_m} (net_{l_m} (t - m)) w_{l_m l_{m-1}}}$$
